# Smoke-Signals
Smoke Signals TC thermocouple and voltage/amperage reader

## Firmware Map
- firmware/SmokeSignals_Receiver/SmokeSignals_Receiver_LoRa_Rev11.ino: System-wide LoRa receiver for Kisew-TC and Waska power node
- firmware/Kisew_TC/sender/Kisew_TC_Sender_LoRa_Rev3.ino: Thermocouple sender node
- firmware/Waska_PN/sender/Waska_PN_A_Rev22_BatRuleAllowsAmps.ino: Voltage / amperage power node sender
